API de DeepL

La API es una interfaz que permite a otros programas informticos enviar textos a nuestros servidores y recibir traducciones de gran calidad. Esto abre la puerta a un nuevo universo de oportunidades para los desarrolladores de todo el mundo: cualquier producto de traduccin imaginable puede crearse ahora empleando la tecnologa de traduccin superior que ofrece DeepL.

Aqu van unos ejemplos:

  • Comunicacin interna: Las grandes compaas pueden integrar nuestra tecnologa de traduccin en sus sistemas de comunicacin. En el mismo instante, obtendrn traducciones de gran calidad. Todo ello garantizando la confidencialidad de la informacin gracias a nuestra garanta de proteccin de datos.
  • Traduccin asistida por ordenador: Las compaas lderes en herramientas de traduccin ya han hecho posible integrar la API de DeepL en sus programas. A partir de ahora, los traductores pueden sacarle el mximo partido al increble poder de las redes neuronales de DeepL en sus herramientas de traduccin.
  • Traduccin en tiempo real: Los gigantes tecnolgicos, as como algunas start-ups han anunciado el desarrollo de auriculares capaces de traducir en tiempo real. Estas traducciones ofrecen solo la calidad de las aplicaciones de traduccin tradicionales. La nueva API de DeepL brinda la ocasin a los desarrolladores de crear traductores en tiempo real empleando la tecnologa de traduccin de DeepL, ms perfecta y con muchos menos errores de traduccin.
  • Traduccin con realidad aumentada: Apunta con la cmara de tu smartphone a un cartel para ver su traduccin adecuada. Una aplicacin as podra desarrollarse con una suscripcin de DeepL Pro.
  • Sistemas operativos: Traduce directamente en tu navegador, en una aplicacin de chat, en tu aplicacin de correo electrnico o en tu procesador de textos. Tan pronto como los desarrolladores integren nuestra tecnologa.

Se puede acceder a la API de DeepL mediante los planes DeepL API Free y DeepL API Pro. El plan DeepL API Free permite a los desarrolladores traducir hasta 500 000 caracteres al mes de manera gratuita. El plan DeepL API Pro ofrece un servicio ampliado: traduccin ilimitada, mxima seguridad de datos y prioridad en la ejecucin de peticiones enviadas a la API.

La siguiente documentacin sirve de referencia para ambos planes. Importante: la direccin de llamada a la API es diferente en cada plan DeepL API; comprueba que ests utilizando la direccin que corresponda.
