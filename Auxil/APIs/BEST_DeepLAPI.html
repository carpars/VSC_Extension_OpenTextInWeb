API de DeepL

La API es una interfaz que permite a otros programas informáticos enviar textos a nuestros servidores y recibir traducciones de gran calidad. Esto abre la puerta a un nuevo universo de oportunidades para los desarrolladores de todo el mundo: cualquier producto de traducción imaginable puede crearse ahora empleando la tecnología de traducción superior que ofrece DeepL.

Aquí van unos ejemplos:

  • Comunicación interna: Las grandes compañías pueden integrar nuestra tecnología de traducción en sus sistemas de comunicación. En el mismo instante, obtendrán traducciones de gran calidad. Todo ello garantizando la confidencialidad de la información gracias a nuestra garantía de protección de datos.
  • Traducción asistida por ordenador: Las compañías líderes en herramientas de traducción ya han hecho posible integrar la API de DeepL en sus programas. A partir de ahora, los traductores pueden sacarle el máximo partido al increíble poder de las redes neuronales de DeepL en sus herramientas de traducción.
  • Traducción en tiempo real: Los gigantes tecnológicos, así como algunas start-ups han anunciado el desarrollo de auriculares capaces de traducir en tiempo real. Estas traducciones ofrecen solo la calidad de las aplicaciones de traducción tradicionales. La nueva API de DeepL brinda la ocasión a los desarrolladores de crear traductores en tiempo real empleando la tecnología de traducción de DeepL, más perfecta y con muchos menos errores de traducción.
  • Traducción con realidad aumentada: Apunta con la cámara de tu smartphone a un cartel para ver su traducción adecuada. Una aplicación así podría desarrollarse con una suscripción de DeepL Pro.
  • Sistemas operativos: Traduce directamente en tu navegador, en una aplicación de chat, en tu aplicación de correo electrónico o en tu procesador de textos. Tan pronto como los desarrolladores integren nuestra tecnología.

Se puede acceder a la API de DeepL mediante los planes DeepL API Free y DeepL API Pro. El plan DeepL API Free permite a los desarrolladores traducir hasta 500 000 caracteres al mes de manera gratuita. El plan DeepL API Pro ofrece un servicio ampliado: traducción ilimitada, máxima seguridad de datos y prioridad en la ejecución de peticiones enviadas a la API.

La siguiente documentación sirve de referencia para ambos planes. Importante: la dirección de llamada a la API es diferente en cada plan DeepL API; comprueba que estés utilizando la dirección que corresponda.

Ejemplo

El siguiente ejemplo muestra una petición básica para traducir el texto “Hello, world!”:

Example Request
curl https://api-free.deepl.com/v2/translate \ 
	-d auth_key=637f2bba-cb50-b69d-680b-a67b8c37ab0f \ 
	-d "text=Hello, world!"  \ 
	-d "target_lang=DE"
Example Response
{
	"translations": [{
		"detected_source_language":"EN",
		"text":"Hallo, Welt!"
	}]
}
